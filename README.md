## ğŸš€ Quick Start

### Prerequisites
- Python 3.11+
- pip package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/thatrasunil/healayur.git
   cd healayur
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure AI Features** (Optional but recommended)
   - The app includes Google Gemini API integration for enhanced chatbot functionality
   - API key is pre-configured for immediate use
   - For production deployment, consider using environment variables

4. **Run the application**
   ```bash
   python app.py
   ```

4. **Access the application**
   - Open your browser and go to `http://localhost:5000`
   - Start analyzing your skin with AI-powered precision!
